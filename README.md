เว็บไซต์คลังข้อสอบและสรุปบทเรียน พัฒนาด้วย Django Framework  
ระบบนี้ช่วยรวบรวมข้อสอบเก่าและสรุปบทเรียน เพื่อช่วยให้นักเรียนทบทวนได้ง่ายขึ้น  
เน้นการค้นหาที่สะดวก ประหยัดเวลา และช่วยเตรียมตัวสอบได้อย่างมีประสิทธิภาพ

---

# วัตถุประสงค์ของระบบ

ระบบนี้ถูกพัฒนาขึ้นเพื่อ

- รวบรวม **ข้อสอบเก่า**
- รวม **สรุปบทเรียน**
- ช่วยให้นักเรียน **ค้นหาข้อมูลได้ง่าย**
- ลดเวลาในการหาข้อมูลก่อนสอบ
- ช่วยให้ผู้ใช้งานสามารถ **แบ่งปันเนื้อหาการเรียนรู้ให้ผู้อื่นได้**

---

# ฟังก์ชันการทำงานของเว็บไซต์

##  Navigation Bar

ด้านบนของเว็บไซต์มี **Navigation Bar** สำหรับเข้าถึงหน้าต่าง ๆ ได้สะดวก ได้แก่

- หน้าแรก (Home)
- รายการทั้งหมด (All Contents)
- เพิ่มข้อมูล (Add Content)

ผู้ใช้สามารถกดเพื่อไปยังหน้าต่าง ๆ ของระบบได้อย่างรวดเร็ว

---

# สำรวจคลังความรู้

ภายในเว็บไซต์มีระบบรวบรวมเนื้อหาความรู้ 2 ประเภทหลัก

## 1️ สรุปบทเรียน

- มีบทเรียนให้เลือกอ่าน
- ผู้ใช้สามารถค้นหาบทเรียนได้ผ่าน **ช่องค้นหา**
- เหมาะสำหรับใช้ทบทวนเนื้อหาก่อนสอบ

---

## 2️ ข้อสอบเก่า

- รวมข้อสอบเก่าจากหลายวิชา
- สามารถเปิดอ่านเพื่อฝึกทำข้อสอบ
- มี **ช่องค้นหา** เพื่อค้นหาข้อสอบที่ต้องการได้ง่าย

---

#  แบ่งปันเนื้อหาใหม่

ผู้ใช้สามารถเพิ่มเนื้อหาใหม่เข้าสู่ระบบได้ เช่น

- สรุปบทเรียน
- ข้อสอบเก่า

### ขั้นตอนการเพิ่มเนื้อหา

1. กรอกข้อมูล
   - รหัสวิชา
   - ชื่อวิชา / หัวข้อสรุป
   - รายละเอียดเนื้อหา

2. เลือกข้อมูลเพิ่มเติม
   - ประเภทเนื้อหา
   - หมวดหมู่วิชา

3. อัปโหลดไฟล์เนื้อหา
   - รองรับไฟล์ **PDF**

4. กด **บันทึกข้อมูล**

หลังจากนั้นเนื้อหาจะถูกเพิ่มเข้าสู่ระบบ

---

# รายการที่เพิ่งอัปโหลดล่าสุด

หน้าเว็บไซต์จะแสดง **รายการที่เพิ่งอัปโหลดล่าสุด**

ผู้ใช้สามารถ

- กด **ดูทั้งหมด (View All)**  
- เพื่อตรวจสอบบทเรียนและข้อสอบใหม่ ๆ ที่ถูกอัปโหลดเข้ามา

ช่วยให้ไม่พลาดเนื้อหาสำคัญ  
และสามารถติดตาม **สรุปบทเรียนและแนวข้อสอบล่าสุดที่เพื่อน ๆ แบ่งปัน**

เพื่อเตรียมตัวสอบได้ก่อนใคร
